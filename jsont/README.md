# jsont

json transformer by using GP Tree


## Development

- Tree Operations


### Run UnitTest

- execute unittest w/ stdout by `nose`.

```sh
$ cd jsont
$ python runtests.py -s
```

- if like to run in watch mode.

```sh
$ pip install pytest-watch
$ ptw
```

### Run Code

- execute in python (requires ver 3.x)

```sh
$ cd jsont
$ python -m gp.op
```
